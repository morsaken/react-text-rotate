# React Text Rotate Usage

So simple to use. First import **ReactTextRotate** and use it in your application.

```
import ReactTextRotate from 'react-text-rotate';

const App = () => {

  return (
    <ReactTextRotate
     texts={['You', 'are', 'awesome']}
     speed={1}
     period={2000}
    />
  );
};
```
